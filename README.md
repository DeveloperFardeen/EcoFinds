# EcoFinds - Sustainable Second-Hand Marketplace

## 🌿 Project Overview
EcoFinds is a modern web application designed to promote sustainability through a user-friendly second-hand marketplace. The platform connects eco-conscious buyers and sellers, helping extend product lifecycles and reduce environmental impact.

## 💻 Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/EcoFinds.git
cd EcoFinds
```

2. Create and activate a virtual environment:
```bash
python -m venv venv
.\venv\Scripts\activate  # Windows
source venv/bin/activate  # Unix/MacOS
```

3. Install dependencies:
```bash
pip install -r requirements.txt
```

4. Set up environment variables:
```bash
set FLASK_APP=run.py
set FLASK_ENV=development
```

5. Initialize the database:
```bash
flask db upgrade
```

6. Run the application:
```bash
flask run
```

## 🌟 Usage

1. Register a new account or login with existing credentials
2. Browse available products by category
3. List items for sale with detailed descriptions and images
4. Add desired items to cart
5. Complete purchases through the checkout process
